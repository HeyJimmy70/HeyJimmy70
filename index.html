Setting up a new Windows machine

  • Click here to download tool to create Windows 10 installation media ISO.
  • Click here to download portable version of Rufus to create USB installation media from Windows ISO.
  • Create new folder "Toolkit" on USB installation media.
  • Create file getPatchMyPC.ps1 PowerShell script to download PatchMyPC. 
    $client = new-object System.Net.WebClient
$source = "https://patchmypc.com/freeupdater/PatchMyPC.exe"
$target = "C:\Users\root\Desktop\PatchMyPC1.exe"
$client.DownloadFile($source,$target)
  • Install Windows 10
